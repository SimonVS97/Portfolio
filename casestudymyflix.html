Case study: myFlix full-stack project

Introduction

Summary

myFlix is a web-app and acts as a library for movies. Users have access to the stored movies and can access movie related information, such as information on the director, the genre and the movie itself. Moreover, users are able to create an account, update their personal data and create a list of favorite movies.

Objective

The goal of this project was to create a full-stack project I can add to my software portfolio. Therefore, the problem I wanted I solve was to build the entire front-end and back-end on my own.

Approach

Back-end

The back-end consists of a RESTful API, that interacts with a NoSQL database. The API was created using Node.js and the Express framework and can be accessed using HTTP methods like GET or POST. The API is hosted on Netlify. The database was created with and is hosted on MongoDB. It also includes user authentication using JWT authorization and user authorization.

Front-end

The front-end is a single-page, responsive application. It consists of a main view, that contains multiple sub-views. The design was created using react bootstrap. Several UI components enable the user to send requests to the server. Furthermore, the responses on the server are rendered in the respective view.

Conclusion

Experiences

This is one of my favorite projects, because it started my excitement about react. Building this web app taught me basic concepts such as how to create modern user interfaces while also introducing me to more advanced concepts like centralizing the state using react redux. All in all it was such a great learning experience.

Duration

It took me about 3 weeks to finish the project.
