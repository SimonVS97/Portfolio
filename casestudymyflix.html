Case study: myFlix full-stack project

Introduction

Summary

myFlix is a web-app and acts as a library for movies. Users have access to the stored movies and can access movie related information, such as information on the director, the genre and the movie itself. Moreover, users are able to create an account, update their personal data and create a list of favorite movies.

Objective

The goal of this project was to create a full-stack project that I can add to my software portfolio. Therefore, the problem I wanted I solve was to build the entire front-end and back-end on my own.

Approach

Back-end

The back-end consists of a RESTful API, that interacts with a NoSQL database. The API was created using Node.js and the Express framework and can be accessed using HTTP methods like GET or POST. The API is hosted on Netlify. The database was created with and is hosted on MongoDB. It also includes user authentication using JWT authorization and user authorization.

Front-end

The front-end is a single-page, responsive application. It consists of a main view, that contains multiple sub-views. The design was created using react bootstrap. Several UI components enable the user to send requests to the server. Furthermore, the responses of the server are dynamically rendered in the respective views.

Conclusion

Experiences

In this project, I wanted to gain experience in full-stack development by building the entire client-side and server-side of a web-app on my own. My idea was to create myFlix, a movie library app.

First, I created the database that stores the data of the movies and the users. Then I set up the web server, that interacts with the database. Moving on to the front-end, I created different views and ensured that the views communicate properly with the web-server and display the data correctly. And with that, the project was finished.

All in all, creating myFlix was such a great learning experience. Building this web app taught me basic concepts such as how to create modern user interfaces while also introducing me to more advanced concepts like client-server-communication. myFlix really started my excitement about react development and encouraged me to keep learning.

Duration

Completing this project took me almost 3 weeks. The reason for this is that I needed spend some time to fully understand the different concepts of React.
